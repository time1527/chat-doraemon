# ref

## 模块

### 微调

数据集：

* 自我认知
* 基础数据：比如，你几岁，你喜欢什么，你讨厌什么
* 对话数据：
  * .mp4抽取
  * 根据经典语录生成问题
* 通过旁白/其他人的信息得到数据：比如，哆啦美为什么比你聪明

### agent

* 道具：
  * 动漫中原有的道具：比如任意门、竹蜻蜓
  * 现代世界的道具：比如文生图

### 语音

* 提取清晰语音，去噪等，使得能够通过输入文本输出特定音色

### 前端

* 暂定gradio/streamlit：结合下已有成功项目的结果再定
